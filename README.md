# jetson-nano-racer

Self-driving car using Jetson Nano, Latrax Rally RC Car, and Machine Learning CNNs.

Hardware: Jetson Nano 4GB Developer Kit with ARM Cortex-A57 CPU

Software: Ubuntu 18.04.6 LTS, Jetpack 4.6.1 SDK, and Python 3.6.9

Camera: Intel RealSense D435i

RC Car: Latrax Rally RC Car

Other hardware: PCA9685, Pololu 4-Channel RC Servo Multiplexer, Fan-4020-PWM-5V, TL-WN725N Wi-Fi Adapter, XBOX Controller

Model architecture: Supports several Resnet variants: ```Resnet18, Resnet34, Resnet50, Resnet101```
