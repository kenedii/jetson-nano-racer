jetracer_full_build.sh: Installs, builds, and sets up the Jetracer and Pytorch python libraries.

xbox_calibrate.py: After all hardware is set up, plug in a USB XBOX controller into the Jetson Nano. This script will allow you to setup the XBOX controller to control the RC car.

car_controller_test.py: After running the xbox_calibrate.py script to setup the XBOX controller to control the RC car, use this script to control the car with the XBOX controller
